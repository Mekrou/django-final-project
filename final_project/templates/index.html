I created three applications relating to the online multiplayer game Overwatch 2. You can add PC accounts at registration, and view their ranks at Rank Tracker. Map Picker was made for custom game lobbies to randomly pick a map for indecisive teams. All data is retrieved from overfast api, which scrapes data off of Blizzard's website, since Overwatch 2 has no official API at the moment. I've left some of my friends and I's battle.net account names in case you do not have any to test.
